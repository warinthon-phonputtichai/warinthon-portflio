Design and creation Program for managing accumulated points to exchange for items or promotions.

SMILE SOCIETY CARD

Smile Society Card is a program for managing points to be exchanged for products or promotions and special privileges, etc. What's special about Smile Society Card is that you can use points to help society. It uses Python language to develop the program and stores data using SQL.
